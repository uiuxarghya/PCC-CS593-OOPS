## Run Locally

### Clone the project

```bash
git clone https://github.com/uiuxarghya/PCC-CS593-OOPS
```

### Go to the project directory

```bash
cd PCC-CS593-OOPS
```

### Go to the specific folder

```bash
cd <Folder Name> ...
```

### Compile & Run the program

- Compile the program

    ```bash
    javac <filename>.java
    ```

- Run the program

    ```bash
    java <filename>
    ```
